## TypeScript + Next + Mantine

developing a project using Next and Typescript.

## Installation

Use a package manager of your choice (npm, yarn, etc.) in order to install all dependencies

```bash
yarn
```

## Usage

Just run `start` script.

```bash
yarn run dev
```
